Uduotis

Faile train.csv yra pradiniai duomenys skirti tyrimui, o faile input.csv yra duomenys skirti prognozavimui. Jums reikia suprognozuoti input.csv faile trkstamas y reikmes, remiantis train.csv faile esania informacija. Btinai pateikite prognozi grafikus.

Sprendimas

Nuskaitome duomenis.

dtrain<-read.csv("https://raw.githubusercontent.com/1vbutkus/Econometrics1/master/task16/train.csv", header=TRUE)
dinput<-read.csv("https://raw.githubusercontent.com/1vbutkus/Econometrics1/master/task16/input.csv", header=TRUE)

plot(dtrain$x, dtrain$y) # pasiirime, kaip atrodo grafikas

plot(dtrain$x[dtrain$color==1], dtrain$y[dtrain$color==1])

Tai yra spalva. Iekome regresijos funkcijos nuo pirmos spalvos.

y1<-dtrain$y[dtrain$color==1]
x1<-dtrain$x[dtrain$color==1]
lm(y1~x1)

Iekome regresijos funkcijos nuo antros spalvos.

y2<-dtrain$y[dtrain$color==2]
x2<-dtrain$x[dtrain$color==2]
lm(y2~x2)
## 
## Call:
## lm(formula = y2 ~ x2)

coef2<-coefficients(lm(y2~x2)) #isaugojame koeficientus

Prognoz

Papildome input failo duomenis.

inputy<-rep(0, times=length(dinput$x))

trainy1<-function(x){coef1[1]+coef1[2]*x} # pagal tiesins regresijos lygt

trainy2<-function(x){coef2[1]+coef2[2]*x}

for(i in 1:length(dinput$x)){
  
  if(dtrain$color[i]==1){
    inputy[i]=trainy1(dinput$x[i])
  }
  
    else{
       inputy[i]=trainy2(dinput$x[i])
       }
}

Nubraiome input duomen grafik.

plot(dinput$x, inputy)
